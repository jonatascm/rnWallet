## Description

Simple react-native wallet connecting to localhost ganache

Developed using:
-EthersJS
-Ganache (Local development)

## Getting Started

First, install dependencies:

```bash
yarn
```

Then, execute ganache and run the app:

```bash
yarn ios
or
yarn android
```
